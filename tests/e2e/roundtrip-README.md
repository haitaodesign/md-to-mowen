# Roundtrip Test: sample.md → Mowen → sample.md

运行 `npx tsx scripts/roundtrip-test.ts > /tmp/roundtrip.md`

与原 `sample.md` diff 后，丢失类别如下：

## 预期有损（Forward 转换设计决定，见 CONCEPT.md §2）

| 元素         | 原始                 | Roundtrip                               | 原因                        |
| ------------ | -------------------- | --------------------------------------- | --------------------------- |
| 标题 H1-H6   | `# 文本`             | `**文本**`                              | 无原生标题，映射为粗体段落  |
| 分隔线 `---` | `---`                | 空行                                    | 映射为空 paragraph          |
| 围栏代码块   | ` ```lang ``` `      | 每行 `` `code` ``                       | 无原生代码块，拆成行内 code |
| 表格         | `\| A \|`            | `![table](mowen://file/uuid)`           | 渲染为 PNG 后上传           |
| 图片路径     | `./assets/img.png`   | `![alt](mowen://file/uuid)`             | 本地路径替换为 fileId       |
| 音频路径     | `./assets/music.mp3` | `![audio: showNote](mowen://file/uuid)` | 同上                        |
| 列表结构     | `- 项目` / `1. 项目` | `• 项目` / `1. 项目`                    | 映射为带前缀文本段落        |

## 预期格式差异（语义等价）

| 元素     | 原始           | Roundtrip      |
| -------- | -------------- | -------------- |
| 斜体     | `_斜体_`       | `*斜体*`       |
| 粗斜体   | `**_粗斜体_**` | `***粗斜体***` |
| 行内代码 | `\`code\``     | `` `code` ``   |

## 预期丢失（AST 层面无法保留）

| 元素           | 原始                    | Roundtrip                | 说明                       |
| -------------- | ----------------------- | ------------------------ | -------------------------- |
| 转义字符       | `\*不是斜体\*`          | `*不是斜体*`             | remark-parser 消费转义符   |
| 多行引用空行   | `> 第一行\n>\n> 第二行` | `> 第一行\n> \n> 第二行` | NoteAtom 用 `\n` 表示换行  |
| 列表缩进精确值 | 多级缩进                | 基本缩进                 | 缩进级别部分保留但精度降低 |
