# 错误码

> 在线文档：<https://mowen.apifox.cn/6688045m0>

---

## 通用错误格式

```json
{
  "code": 40001,
  "reason": "错误描述"
}
```

---

## 已知错误码

| code | 含义 | 处理建议 |
|---|---|---|
| `40001` | 参数错误 | 检查请求体字段 |
| `40301` | 未授权 / API Key 无效 | 检查 `Authorization` header |
| `40302` | 会员已过期 | 用户需续费墨问会员 |
| `40303` | API Key 已重置，当前 Key 失效 | 从环境变量重新获取最新 Key |
| `42901` | 频率超限 | 降低调用频率，见频率限制 |
| `50001` | 服务器内部错误 | 稍后重试，建议指数退避 |

> 注：错误码列表从 mowen.sh 和 skill 代码反推，完整列表以 <https://mowen.apifox.cn/6688045m0> 为准。

---

## 频率限制错误

当触发频率限制时，返回 HTTP 429 或 `code: 42901`。

建议策略：
- 实现指数退避重试（初始 1s，最大 30s，最多 3 次）
- 对 `upload/prepare` 操作缓存 fileId，避免重复上传同一文件
